ItemConsumableSetEnergy

Syntax

ItemConsumableSetEnergy(item_id, amount);

Returns "true", if successful, otherwise it will crash.

Argument Type Description
item_id String The name of the item to adjust.
amount Int Amount of Energy/Hunger the player gets.

Energy in this case is Hunger which allows the player to carry more while they are not hungry.

This will allow you to change the amount that the players Hunger is changed once it is fully consumed.

Please note: This could be a positive or negative number. A negative number will make the player more hungry and a positive number will restore the players energy level.

Example:

ItemConsumableSetEnergy("Pint of Dave", 20)

The above code will make it so when "Pint of Dave" is consumed the player will gain 20 hunger and also lose any debuffs they may have from having low hunger.
